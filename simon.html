  • How to play
  • Press any keyboard key to begin.
  • A color will flash.
  • Remember the order as new colors get added.
  • Click the buttons in the same order.
  • Get it right? You move to the next level.
  • One new color gets added each time.
  • Click the wrong color? The game ends.
  • Press a key to start again or click "Reset."
